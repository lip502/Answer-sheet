# MATLAB答题卡识别（GUI，论文）

#### 介绍
本课题为基于MATLAB的不变矩答题卡识别。可识别学号，学科，答案，并进行分数统计，以及判断是否及格。后台可设置标准excel答案。经过灰度变换形成灰度图像、二值化处理、图像滤波、边缘检测、hough变换等预处理。带GUI交互式界面。
